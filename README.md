## Description

This repository contains the code for a backend API developed as part of a [coding challenge](https://bossabox.notion.site/Back-end-0b2c45f1a00e4a849eefe3b1d57f23c6). 

The API is built using the [Nest](https://github.com/nestjs/nest) framework, a powerful, scalable framework for building efficient and reliable server-side applications.

The API provides a set of endpoints for managing tools, including operations for creating, updating, retrieving, and deleting tools. Each tool has properties such as name, link, description, and tags.

This project demonstrates the use of various technologies and practices such as TypeScript, Object-Relational Mapping (ORM), Docker, TypeORM, PostgreSQL, Swagger, environment variable management, and more.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```
